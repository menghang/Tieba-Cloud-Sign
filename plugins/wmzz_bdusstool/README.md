#获取BDUSS工具插件
这是适用于 贴吧云签到的 获取BDUSS工具插件                       
下载ZIP直接安装请注意去掉压缩包内文件夹名字的-master       
###插件说明
激活此插件后将在 [手动绑定] 页面显示BDUSS工具获取BDUSS的方法并提供工具下载      
获取工具由 @mokeyjay 制作             
插件由 @kenvix 制作
